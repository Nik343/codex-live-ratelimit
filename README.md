## Codex live ratelimit

A minimal Python 3.10+ script that asks `codex app-server` for the live 5-hour and weekly rate limit of any given Codex profile. Inspired by [xiangz19/codex-ratelimit](https://github.com/xiangz19/codex-ratelimit).

### Usage

Run `python codex_live_ratelimit.py`. Optionally pass `--input-folder /path/to/.codex` or `--input-folder /path/to/.codex/sessions`.
